A WordPress plugin for managing redirects in bulk.

== Description ==
Bulk Redirector allows you to manage URL redirections in bulk using CSV files.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/csv-redirector` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
